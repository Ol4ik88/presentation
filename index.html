Pattern definition

A design pattern or programming pattern

is a typical way of solving common problems in software development.

Pattern MVC

MVC (Model-View-Controller)

is a fundamental design pattern whose purpose is to separate user interface logic from programming logic

The concept of the MVC

  • Model: Manages data and business logic
  • View: Handles layout and display
  • Controller: Routes commands to the model and view parts
